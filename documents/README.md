| `abstracts/` | Official abstract, *Gravitational Storage-Regeneration Systems* | 2025–26 |

| `research-plans/Research_Plan_2025-26_Regional.docx` | Regional research plan for the four-system comparison (simulation procedure + prototype procedure) | 2025–26 |
| `concept-paper/gravity_battery_concept_paper.tex` | LaTeX concept manuscript, *Comparative Design of Four Gravity-Based Energy Storage Architectures* by A. S. Khan — **explicitly a demonstration draft with placeholder values**, prepared as a template for the future technical paper | 2026 |
| `notes/` | Working notes: motor selection research, math formulas, system ideas, board image citations, pacing log, registration answers, presentation script drafts (the 2026 design task list — incl. the Virginia Tech elevator-dimensions reference — is in `archive/development-notes/`) | 2024–26 |
